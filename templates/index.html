Добавить ещё одну форму

Чтобы получить максимальную оценку по проекту, нужно две страницы, которые показывают контент, а две страницы, которые принимают контент. Одна страница с POST-запросом создана — это форма сбора терминов. Добавьте ещё одну форму и view, обрабатывающий её!

Разнообразить контент

Этот проект — про математику. В условии проекта в курсе — иностранные языки. Измените содержимое страниц, чтобы они были полезны в контексте изучения иностранного языка (несколько примеров есть в описании проекта).

Изменить вёрстку и/или дизайн-систему

Этот проект использует Bootstrap. Попробуйте другие дизайн-системы, например:

  • Materialize
  • Semantic UI
  • Foundation

Если же вам хочется оставить Bootstrap — измените вёрстку! Добавьте картинок, замените стили — словом, пробудите внутреннего художника.
